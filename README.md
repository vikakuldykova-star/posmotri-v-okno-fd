# Посмотри в окно

Веб‑приложение, которое позволяет искать и просматривать видео по городам и времени суток.
Проект выполнен с использованием **HTML, CSS и JavaScript**.

## Функционал
- Поиск видео по названию города.
- Фильтрация по времени суток (утро, день, ночь).
- Просмотр выбранного видео в плеере.
- Список карточек с результатами поиска.
- Кнопка «Показать ещё» для подгрузки дополнительных видео.
- Кастомные элементы интерфейса: чекбоксы, скроллбар, стилизованные кнопки.

## Технологии
- **HTML5** — семантическая разметка.
- **CSS3** — стилизация, кастомный скроллбар, адаптивные элементы.
- **JavaScript (ES6+)** — логика поиска, работа с шаблонами карточек, обработка событий.
- Шрифты: *Fira Sans Condensed*, *Oswald*.
